Encabezados Estos elementos representan títulos para sus secciones. 'h' viene de la palabra Heading que significa Encabezado. Puedes visitar el estándar de HTML para ver ejemplos: https://html.spec.whatwg.org/multipage/sections.html#the-h1,-h2,-h3,-h4,-h5,-and-h6-elements --- PRO TIP: -- En un documento HTML, es recomendable tener solo un encabezado principal h1. Los motores de búsqueda utilizan estos encabezados para comprender la estructura y el contenido principal de tu página. Al mantener un único h1 claro y relacionado con el tema central, aumentas las posibilidades de que tu sitio web obtenga un mejor posicionamiento en los resultados de búsqueda. Esta práctica de optimización se conoce como SEO (Search Engine Optimization).
